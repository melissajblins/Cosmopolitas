# Cosmopolitas
Um presente de aniversário digital feito com Bootstrap.

## Seções do site Cosmopolitas
1. Navbar
2. Carrossel
3. Jumbotron: Mensagem
4. Grid: Fotos e Spotify Embed
5. Tabela
6. Cards: Álbum de fotos
7. Input Group: Lista
